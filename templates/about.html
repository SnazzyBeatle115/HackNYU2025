About This Project

This is a Flask-based web application with an integrated machine learning component. The project demonstrates a clean separation of concerns with a dedicated ML folder for all machine learning-related code.

Project Structure

├── app.py                  # Main Flask application
├── templates/              # HTML templates
│   ├── base.html          # Base template
│   ├── index.html         # Home page
│   └── about.html         # About page
├── static/                 # Static files
│   ├── css/               # Stylesheets
│   ├── js/                # JavaScript files
│   └── images/            # Images
├── ml/                     # Machine Learning folder
│   ├── __init__.py        # Package initialization
│   ├── model.py           # ML model implementation
│   └── utils.py           # ML utilities
└── requirements.txt        # Python dependencies

Technologies Used

  • Flask: Lightweight web framework for Python
  • NumPy: Numerical computing library
  • Machine Learning: Modular ML implementation (ready for scikit-learn, TensorFlow, PyTorch, etc.)

Getting Started

  1. Install dependencies: pip install -r requirements.txt
  2. Run the application: python app.py
  3. Visit http://localhost:5000 in your browser
